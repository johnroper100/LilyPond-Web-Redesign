Mac OS X

Csomagok

Letöltés

  • MacOS X x86: LilyPond 2.18.2-1 Intel processzorokra (ha bizonytalan vagy afelől, hogy melyiket töltsd le, ezt válaszd)

    A grafikus felület nem működik Mac OS X 10.7 Lion alatt! Ha a grafikus felületet szeretnéd használni, töltsd le a legújabb fejlesztői verziót!

  • MacOS X PPC: LilyPond 2.18.2-1 PowerPC processzorokra (régi Apple gépekben találhatóak)

    Csak 2006 előtti Apple gépeken fut!

Telepítés

Csak kattints duplán a letöltött fájlra a kitömörítéséhez, majd a kicsomagolt LilyPond.app alkalmazást húzd be az Applications könyvtárba.

Eltávolítás

Töröld a LilyPond.app alkalmazást.

Parancssori futtatás

Figyelem: Ha a grafikus felület megfelel, ezeket az utasításokat nyugodtan figyelmen kívül hagyhatod.

Írd be a könyvtáradban található ‘.profile’ fájlba a következő sort: 

export PATH=/Applications/LilyPond.app/Contents/Resources/bin:$PATH

Jogi tudnivalók

Minden logó és termékábrázolás szerzői joggal védett.
